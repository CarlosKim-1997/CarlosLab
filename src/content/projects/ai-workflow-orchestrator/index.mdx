## 무엇을 하나요

고객 이메일이 들어오면 의도·담당 부서·우선순위를 분류하고, Jira 티켓·Slack 알림·회신 메일까지 자동으로 처리합니다.

## 왜 만들었나요

컨설팅 PoC에서 “분류에 2시간 → 5분” 같은 흐름을 한 화면에서 보여주려고 만들었습니다.

## 어떻게 동작하나요

Streamlit UI → 분석 → 워크플로우 생성 → 플러그인 실행 순입니다. `@register_integration`으로 새 연동을 추가합니다.

## 설계 메모

- 플러그인 구조: SAP·ServiceNow 같은 연동을 코어 수정 없이 추가
- 시뮬레이션 모드: API 키 없이도 데모 가능
- LLM 출력을 JSON으로 받아 downstream에 바로 연결

## 어려웠던 점

데모용 UX와 실제 API 연동 사이를 맞추는 것 — 화면에 보이는 것과 실제로 도는 것의 간격 조절
