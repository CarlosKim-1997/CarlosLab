## 무엇을 하나요

“오늘 뭐 먹지?”를 짧은 질문으로 좁힌 뒤, 메인 1개 + 대안 2개만 보여 줍니다.

## 왜 만들었나요

정확한 AI 추천보다, 고민 시간을 줄이는 게 목표였습니다.

## 어떻게 동작하나요

상황 선택 → 질문 → filters / scoring / variety → 결과. 지도·배달 앱으로 넘어갈 때만 광고가 나옵니다.

## 설계 메모

- 결과 개수 고정 (더 보기 없음)
- 최근 먹은 메뉴는 페널티만, 완전 제외는 안 함
- Supabase 키 없어도 앱은 돌아감

## 어려웠던 점

캐릭터 톤(담백·무심)을 UI 카피와 코드 전반에 맞추는 것
